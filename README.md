## Описание

В ходе лабораторной работы была изучена работа с DOM и событиями в JavaScript:
- Что такое DOM и как JavaScript взаимодействует со страницей
- Поиск элементов через getElementById и querySelector
- Изменение текста и стилей элементов через JS
- Обработка событий с помощью addEventListener
- Работа с полем ввода input и получение значений через .value
- Валидация данных формы (проверка пустых полей, диапазона чисел)
- Динамическое создание и удаление элементов через createElement и appendChild
- Делегирование событий

## Структура проекта

- `index.html` — основная HTML-страница DOM
- `main.js` — основной js файл
- `dynamicElements.html` — страница для демонстрации динамического создания элементов
- `dynamicElements.js` — скрипт динамического списка пользователей
- `style.css` — стили
- `README.md` — описание
- `img/` — скрины
